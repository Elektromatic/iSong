C----------------------------------------------------------------------------
C  			          iSong                
C----------------------------------------------------------------------------
C
C  This Max 8 patch creates "generative music" in conjunction with an 
C  electronic synthesizer.  The generative music creates an aletoric song
C  in conjuction with an Elektron Digitakt synthesiser.
C
C  Please see https://youtu.be/M9mbs0VJo_0  for an example of the use of this
C  Max 8 patch.
C
C----------------------------------------------------------------------------
C  Requirements:   Max 8  <https://cycling74.com/>
C----------------------------------------------------------------------------
C
Usage:  The file iSong.maxpat is the parent patch. The other abstractions are
        used by the parent patch.

Contributing: Any contributions to this project are welcome.

Acknowledgements: I'd like to thank Judy for her continued support.
